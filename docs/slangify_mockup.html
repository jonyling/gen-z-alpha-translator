🗣️ Send as Slang / English

Each person types and sends in their own language — jamie in slang, Alex in English. The little “Send as” switch lets either of them flip if they want.

🔁 Auto-translate (top-right)

Turn it on and every incoming message is translated into your language, shown right under the original. Off = read it exactly as sent.

🧠 Won’t force it

If a message isn’t clear, the model abstains instead of inventing a translation — the same “I’m not sure” behavior built into the tool.
