✨ Slangify (sender)

Toggle on and the sender’s plain English is rewritten into current slang before it sends. Toggle off to see exactly what they typed.

🔁 Auto-translate (receiver)

Incoming slang gets a plain-English translation underneath, automatically. No guessing what “Ohio drip is cooked” means.

🧠 Won’t force it

If a message isn’t clear, the model abstains instead of inventing a translation — the same “I’m not sure” behavior built into the tool.
